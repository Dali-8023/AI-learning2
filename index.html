第1章：什么是生成式AI？

第一周 学习时间：2小时 知识点：12个

学习目标

  • 理解生成式AI与传统AI的根本区别
  • 掌握生成式AI的核心概念和术语
  • 了解生成式AI的历史发展和现状
  • 能够识别生成式AI的典型应用场景

1.1 生成式AI的核心概念

传统AI（判别式AI）
  • 功能：分类、识别、预测
  • 工作方式：从数据中学习模式，然后对新的输入进行分类
  • 典型应用：垃圾邮件检测、图像分类、语音识别
  • 输出：标签、类别、概率
  • 例子：识别猫的照片为"猫"
生成式AI
  • 功能：创造、生成、合成
  • 工作方式：学习数据的分布，然后生成新的、类似的数据
  • 典型应用：文本生成、图像创作、音乐合成
  • 输出：新内容（文本、图像、代码等）
  • 例子：生成一张全新的猫的照片
简单比喻

想象一下两种不同的厨师：

  • 传统AI厨师：品尝一道菜，告诉你这是什么菜系、用了什么调料（分类）
  • 生成式AI厨师：品尝很多菜后，创造出全新的、从未有过的菜品（创造）

1.3 关键术语解析

大语言模型(LLM)

基于海量文本数据训练的大型神经网络模型，能够理解和生成人类语言

例子：GPT-4, Claude, LLaMA
参数(Parameters)

模型中可调整的数值，决定了模型的行为和性能

类比：人类大脑的神经连接数量
提示(Prompt)

用户输入给AI的指令或问题，引导AI生成特定内容

例子："写一首关于春天的诗"
生成(Generation)

AI根据输入创建新内容的过程

过程：输入→AI处理→新内容输出

1.4 生成式AI的应用场景

文本生成
  • 创意写作：小说、诗歌、剧本
  • 商务写作：邮件、报告、营销文案
  • 内容创作：博客文章、社交媒体内容
  • 翻译与总结：文档翻译、文章摘要

  • 艺术创作：数字绘画、概念艺术
  • 设计辅助：Logo设计、UI界面
  • 照片编辑：背景替换、风格转换
  • 3D建模：3D模型生成、纹理创建
代码生成
  • 编程辅助：代码补全、bug修复
  • 算法实现：根据描述生成代码
  • 代码转换：Python转JavaScript等
  • 文档生成：根据代码生成说明文档

  • 音乐生成：旋律创作、和声生成
  • 语音合成：文本转语音、语音克隆
  • 视频生成：短视频创建、特效添加
  • 游戏开发：游戏关卡、角色设计

本章要点总结

  • 生成式AI的核心是"创造"而非"识别"
  • 大语言模型(LLM)是当前生成式AI的主要形式
  • 生成式AI已应用于文本、图像、代码、音乐等多个领域
  • AI的"智能"来自于海量数据和复杂的模型参数
  • 理解AI的能力边界同样重要

第一周 学习时间：3小时 知识点：18个

2.1 LLM到底是什么？

核心本质：概率文本生成机

大语言模型的本质是一个基于概率的文本生成系统。它通过分析海量文本数据，学习语言中的统计规律，然后根据给定的文本预测下一个最可能出现的词。

人类思维的类比

想象你在玩文字接龙游戏：

  • 给定："今天天气真..."
  • 你会想到："好"、"不错"、"晴朗"等词
  • 你选择哪个词取决于你以前听过的类似表达
  • LLM做的是类似的事情，只是规模更大、更复杂
技术视角
# LLM的简化工作流程
def predict_next_word(context):
    # 1. 将输入文本转换为数字表示（词嵌入）
    embeddings = convert_to_embeddings(context)
    
    # 2. 通过神经网络层处理
    hidden_states = process_through_layers(embeddings)
    
    # 3. 计算下一个词的概率分布
    probabilities = calculate_probabilities(hidden_states)
    
    # 4. 选择概率最高的词
    next_word = select_highest_probability(probabilities)
    
    return next_word

2.2 LLM的关键组件

训练数据

作用：模型的"知识库"

规模：通常是TB级别的文本数据

来源：网页、书籍、论文、代码等

模型参数

作用：存储学习到的模式

规模：从数百万到上万亿

功能：权重和偏置，决定输入如何转换为输出

神经网络架构

类型：Transformer架构

层数：数十到数百层

功能：处理和转换输入信息

训练算法

方法：自监督学习

任务：预测被遮盖的词

优化：梯度下降和反向传播

2.3 LLM的能力层次

层级1：基本语言理解
  • 语法理解：识别句子结构
  • 词汇理解：知道单词的含义和用法
  • 基本语义：理解简单句子的意思
层级2：上下文理解
  • 指代消解：理解代词指代的对象
  • 多轮对话：记住对话历史
  • 隐含意义：理解言外之意
层级3：推理与创造
  • 逻辑推理：进行简单逻辑推断
  • 知识应用：应用学到的知识解决问题
  • 内容创作：生成原创内容
层级4：复杂任务处理
  • 多步骤推理：解决需要多步思考的问题
  • 代码生成：编写和调试代码
  • 复杂分析：分析复杂数据并得出结论
